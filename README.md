# Setup and run

## Install tools

1. Docker Desktop
2. Visual Studio Code

Make sure Docker Desktop is running

## Get code

```bash
git clone https://github.com/Lucian-Adrian/globant-practice-2025
```

## Run

```bash
docker-compose up --build
```

Services:
- Database: port 5432
- Backend: port 8000
- Frontend: port 3000

## Stop

```bash
Ctrl + C
docker-compose down
```
